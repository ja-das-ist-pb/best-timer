Game Instructions

Local Mode:

A single or multiplayer mode that player(s) will use the stop watch to approach the target time. The cloest one win.

LAN Mode

Players who connect the same wifi can player together.

At first, every device will recieve a target time, each player will need to click their device as a stop watch to reach the target time. After every player finish, the closest one win.

Internet Mode

Same as the LAN Mode, but players can play online.

How to use the stop watch

Stop Watch will start when player firstly click on the device and end via the second click.
